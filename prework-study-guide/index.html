HTML

  • The head element contains information about the webpage.
  • Most of the content on the head element is invisible for the users.
  • The title is the only thing that is visible on the head element for the users.
  • The head element also contains metadata, metadata is not visible for users.
  • The body element represents the visible content shown to the user.
  • HTML is used to create the structure of a webpage.
  • The structure of a webpage. It is made up of elements which work together to provide
    content and the basic framework for all pages on the web.
  • The (text elements, image elements, header elements and list elements) are commonly used to
    add content to pages.
  • The (h1, h2, h3, h4, h5, and h6) elements represent the level of heading a given text block.
  • When Using the (h1, h2, h3, h4, h5, and h6) elements, you must use them in order so the
    page become accessible to users who access the page using screen readers.
  • The Img element can contain various attributes but the (src) attribute is the most important
    because it defines the location of the image.
  • The (a) element which stands for anchor, creates links to the same webpage or other webpages.
  • HTML uses the (id attribute) as a way to identify elements and select them, this become very
    useful when working with CSS and JavaScript.
  • The (alt) attribute contains a text string that describes the appearance and functionality of an image,
    the (alt text) is important for screen readers and other assistive technologies. (Alt text) can also improve
    search engine rankings for your website.
  • The (footer) appears at the bottom of the page and usually contains the author, copyright, contact, sitemap
    and navigation tools of the web page.

CSS

  • The fist thing you have to do when working with CSS is to link your CSS code sheet to the file you
    are working on.
  • CSS is used to add styles to your web page.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • If you want to add style to several selectors at the same time use commas
  • When you want to add style to specific parts of your html code you can use the
    class selector and give an ID to the specific part of the code so the CSS document affects just what you want to be affected.
  • Common used declarations for CSS files: link, margin, padding, width, height, backgroud-color, color, text-align...
  • Commun used selectors for CSS: H1, H2, H3, H4, H5, H6, * (it affects everithing shown in the page),
    headeer, footer, img, ul, p...
  • In order for the CSS to know that you are about to add style to an ID, you have to add a . to the
    beggining of the selector. Example; .card { color #888888 }

GIT

  • Git is a version control system used for tracking changes in computer files.
  • The first thing you may do when first working with Git, is to set it up by configuring your user name,
    configuring your email and connecting your HSS Key with Github.
  • The following git commands are some of the most used; git add, git commit, git status, git push, git pull,
    git clone, git checkout and git config.
  • The following commands are some of the most use to navigate and interact through Gitbash; cd, pwd, mkdir and --global.
